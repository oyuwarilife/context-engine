# Context Engine

AIエージェントに「自分」を正しく伝えるためのコンテキスト管理ツール。

## 🔗 デモ

**[Context Engine を使ってみる →](https://oyuwarilife.github.io/context-engine/)**

## 概要

AIに毎回同じ自己紹介をしていませんか？

Context Engineは、あなたのプロフィール・ブランド情報・発信スタイルなどを構造化して管理し、AIエージェントにそのまま渡せる形で出力するWebアプリです。

## 主な機能

- **コンテキストの構造化管理** — プロフィール、ブランドポジション、SNS発信スタイルなどをテンプレートから作成
- **カテゴリ分類** — private / work / public でコンテキストを整理
- **AI向けエクスポート** — CLAUDE.md や System Prompt にそのまま貼れる形式で出力
- **ブラウザ完結** — データはローカルストレージに保存。サーバー送信なし

## 使い方

1. [Context Engine](https://oyuwarilife.github.io/context-engine/) にアクセス
2. テンプレートからコンテキストを作成（自己紹介、SNSプロフィールなど）
3. 内容を編集・カスタマイズ
4. エクスポートしてAIツールに貼り付け

## 技術構成

- HTML / CSS / JavaScript（単一ファイル）
- 外部依存なし（フレームワーク不使用）
- GitHub Pages でホスティング
